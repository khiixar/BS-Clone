# BlankSaga Clone - Static Website

This is a static clone of the BlankSaga website for testing purposes.

## Files Included:
- index.html (Home page)
- shop.html (Shop page with all products)
- about.html (About Us page)
- faq.html (FAQ page with accordion)
- contact.html (Contact page)
- reviews.html (Reviews page - NO DUPLICATES)
- product-*.html (8 individual product detail pages)
- styles.css (Global styles)
- script.js (JavaScript for age gate, FAQ, and size selector)
- images/ (8 generic product images)

## How to Use:
1. Extract all files to a folder
2. Open index.html in your web browser
3. Navigate between pages using the menu
4. Click on any product to view its detail page

## Features:
✅ Age verification modal (appears on first visit)
✅ Responsive navigation with dropdown
✅ Clickable product cards that link to detail pages
✅ Individual product detail pages with size selector
✅ Generic product images (colored placeholders)
✅ FAQ accordion functionality
✅ Contact form (static, no backend)
✅ Customer reviews section (NO DUPLICATES)
✅ Footer with newsletter signup (static)

## Product Pages:
Each product has its own detail page with:
- Large product image
- Full description
- Size selector (S, M, L, XL, XXL)
- Add to cart button (static)
- Product details list

## Notes:
- This is a static clone with no backend functionality
- Forms do not submit data
- Cart/Account/Wishlist links are placeholders
- Product images are generic colored placeholders
- Age gate uses localStorage to remember verification
- All products link to their own detail pages

## Customization:
- Replace product images in /images folder with your own
- Update product names, prices, and descriptions in HTML files
- Modify colors in styles.css :root variables
- Add actual form handling with JavaScript or backend service
